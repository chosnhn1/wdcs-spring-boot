# 스프링 부트 프로젝트 구조

* `com.mysite.sbb` 패키지
  * 소스
* `AppnameApplication.java` 파일
  * 프로그램의 시작을 담당
* `src/main/resources`
  * HTML, CSS, JS, 환경 파일의 저장 공간
* `templates`
* `static`
* `application.properties` 파일
  * 프로젝트의 환경설정
* `src/test/java`
  * 테스트 코드 (JUnit, Spring boot 테스트 도구 등)
* `build.gradle`
  * Gradle의 환경 파일
